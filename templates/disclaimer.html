Welcome to Instant OpenHIE!

A little disclaimer first...

Do not use this app in production or with sensitive data.

This is an experiemental desktop app to help with testing, demos, and training, and not an official project of OpenHIE. This app:

  • Creates files in the home directory to store things. This includes a copy of the source code from github.com/openhie/instant These are located in .instant/
  • Queries, starts, and stops containers in Docker and Kubernetes. Only the 'instant' workflows are affected.

This app will contain bugs. Please report issues here.
